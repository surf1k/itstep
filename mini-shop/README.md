# Mini Shop Landing Page

A responsive, single-page e-commerce concept built with React and Vanilla CSS.

## Features
- **Dynamic Hero Gallery:** Automatic image transitions in the header.
- **Product Grid:** Responsive layout featuring 6 curated products.
- **Functional Shopping Cart:** Side drawer for managing quantities and calculating total prices.
- **Responsive Design:** Mobile, tablet, and desktop optimized.

## Tech Stack
- React
- TypeScript
- Vite
- Vanilla CSS

## Getting Started
1. Install dependencies: `npm install`
2. Run development server: `npm run dev`
3. Build for production: `npm run build`
